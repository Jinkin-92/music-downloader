## 🎯 使用指南

### 基本功能

1. **单曲搜索和下载**
   - 输入歌曲名或歌手
   - 选择音乐源（QQ音乐、网易云音乐、酷狗、酷我）
   - 点击"Search"
   - 选择结果，点击"Download"

2. **批量下载**（如果功能可用）
   - 切换到"批量下载"标签页
   - 输入歌曲列表（每行一首）
   - 配置搜索选项
   - 点击"批量搜索"
   - 选择要下载的歌曲
   - 点击"批量下载"

3. **自定义下载路径**
   - 菜单: 文件 → 选择下载路径
   - 选择目标文件夹
   - 后续下载将保存到该路径

### 高级功能

1. **切换匹配结果**（批量下载）
   - 在批量结果表格中点击"切换"按钮
   - 查看不同源的搜索结果
   - 选择最合适的版本

2. **重搜未匹配歌曲**
   - 点击"重搜"按钮
   - 使用不同源重新搜索

## 🔧 故障排查

### 问题1: 应用无法启动

**症状**: 双击批处理文件显示乱码或错误

**解决方案**:
```bash
# 使用Python脚本启动
python run_app.py
```

### 问题2: 提示Python未找到

**症状**: `'python' 不是内部或外部命令`

**解决方案**:
1. 安装Python 3.7+
2. 安装时勾选"Add Python to PATH"
3. 或使用完整路径: `C:\Python313\python.exe run_app.py`

### 问题3: 提示PyQt6或musicdl未安装

**症状**: `ModuleNotFoundError: No module named 'PyQt6'`

**解决方案**:
```bash
pip install PyQt6 musicdl
```

### 问题4: GUI窗口显示乱码

**症状**: 窗口标题或内容显示为方框或问号

**解决方案**:
1. 设置系统区域为中文（中国）
2. 或使用英文界面（功能不受影响）

### 问题5: 搜索无结果

**症状**: 搜索后表格为空

**可能原因**:
- 网络连接问题
- 关键词过于生僻
- 音乐源限制

**解决方案**:
- 检查网络连接
- 尝试不同关键词
- 切换不同音乐源
- 使用批量下载功能（多源搜索）
