# 音乐下载器

一个基于 Python、PyQt6 和 musicdl 的**桌面音乐下载工具**，提供简洁的图形界面进行音乐搜索和批量下载。

## 截图

![界面预览](docs/screenshot.png)

## ✨ 主要功能

### 🔍 多平台搜索
- 支持 **QQ音乐**、**网易云音乐**、**酷狗音乐**、**酷我音乐**
- 实时显示搜索结果（歌曲名、歌手、专辑、大小、时长、来源）
- 可选择性勾选要下载的音乐源

### ✅ 批量勾选下载
- **复选框勾选**：勾选想要下载的歌曲
- **全选/反选**：一键全选或反选所有结果
- **批量下载**：一次性下载多首勾选的歌曲
- **智能确认**：下载前显示勾选数量，避免误操作

### 📊 实时进度显示
- 进度条显示下载进度
- 状态栏显示当前操作和结果统计
- 详细日志记录（保存到 `logs/app.log`）

### 🎯 快捷操作
- **右键菜单**：提供丰富的批量操作选项
- **表头点击**：点击复选框列表头快速全选/取消全选
- **双击下载**：选中歌曲后可直接右键下载

## 📦 环境要求

- **Python**: 3.7+ （推荐 3.10+）
- **操作系统**: Windows / macOS / Linux
- **网络**: 需要能访问音乐平台 API

## 📖 使用说明

### 搜索音乐

1. **选择音乐源**：在顶部勾选想要搜索的平台（默认全选）
2. **输入关键词**：在搜索框输入歌曲名、歌手或关键词
3. **点击搜索**：点击"Search"按钮开始搜索
4. **查看结果**：搜索结果会显示在下方表格中

### 下载音乐

#### 方法一：勾选下载（推荐）

1. **勾选歌曲**：点击表格第一列的复选框，勾选想下载的歌曲
2. **批量选择**：
   - 点击表头的"☐"可以**全选/取消全选**
   - 右键 → "Invert Selection"可以**反选**
3. **下载勾选**：右键 → "Download Checked (N)"（N为勾选数量）
4. **确认下载**：在弹出的对话框中确认下载

#### 方法二：快速下载

- **下载选中行**：单击某一行（高亮显示），右键 → "Download Selected"
- **下载全部**：右键 → "Download All Results"

### 其他操作

- **复制歌曲信息**：右键 → "Copy Song Info"（格式：歌曲名 - 歌手）
- **全选**：右键 → "Select All"
- **取消全选**：右键 → "Uncheck All"

## ⚙️ 配置说明

### 下载目录

默认下载目录：`musicdl_outputs/`（在项目根目录下自动创建）

文件命名格式：
```
{歌曲名} - {歌手}.{扩展名}
```

### 支持的音频格式

- **OGG** (QQ音乐、网易云)
- **FLAC** (无损音质，部分平台支持)
- **MP3** (部分平台支持)

格式和音质取决于音乐平台的提供的资源。

## 🔧 常见问题

### Q: 搜索不到结果？

**A**: 尝试以下方法：
1. 检查网络连接是否正常
2. 尝试使用不同的关键词（歌曲名或歌手名）
3. 切换不同的音乐源

### Q: 下载失败？

**A**: 可能的原因：
1. **网络问题**：检查网络连接
2. **版权限制**：部分歌曲可能有版权保护
3. **平台限制**：某些平台可能需要登录或VIP

### Q: 下载的文件音质如何？

**A**: 音质取决于音乐平台提供的资源。一般来说：
- 网易云：标准音质
- QQ音乐：标准音质
- 酷狗/酷我：标准音质

部分歌曲可能提供高品质或无损音质（需要平台支持）。

### Q: 程序闪退或无法启动？

**A**: 检查：
1. Python 版本是否为 3.7+
2. 是否安装了所有依赖：`pip install -r requirements.txt`
3. 查看日志文件 `logs/app.log` 获取详细错误信息

### Q: Windows 下提示"找不到 Qt 平台插件"？

**A**: 这是 PyQt5 的问题。本项目已升级到 PyQt6，确保安装了 PyQt6：
```bash
pip uninstall PyQt5
pip install PyQt6
```
