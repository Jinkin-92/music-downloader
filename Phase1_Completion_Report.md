## 🎉 用户使用指南

### 基本使用流程

1. **启动应用**
   ```bash
   python launcher.py
   ```

2. **切换到"批量下载"Tab**

3. **输入歌曲列表**
   ```
   告白气球 - 周杰伦
   没关系 - 容祖儿
   七里香 - 林俊杰
   ```

4. **配置匹配模式**
   - **严格模式(≥90%)**: 只显示高质量匹配
   - **标准模式(≥60%)**: 平衡质量和数量 (默认)
   - **宽松模式(≥40%)**: 显示更多候选

5. **批量搜索**
   - 点击"Batch Search"
   - 等待搜索完成

6. **查看结果**
   - 表格显示匹配结果
   - 相似度颜色编码（绿/黄/红）
   - 低于阈值的用斜体标记

7. **切换模式**（可选）
   - 在匹配设置区域切换模式
   - 表格实时更新
   - 自动选择最佳候选

8. **下载**
   - 勾选想要的歌曲
   - 点击"Download Selected"

## 🔧 技术亮点

### 1. 智能候选选择
当当前匹配低于阈值时，自动选择最佳候选：
```python
best_candidate = song_match.auto_select_best_within_threshold(0.90)
if best_candidate:
    song_match.switch_to_candidate(best_candidate, MatchSource.AUTO_FILTERED)
```

### 2. 渐进式UI设计
- 默认隐藏高级选项
- 三种模式快速选择
- 自定义阈值可选

### 3. 偏好持久化
使用QSettings自动保存用户选择：
```python
settings.setValue("match_mode", self.current_match_mode.value)
settings.setValue("custom_threshold", self.current_threshold)
```

### 4. 实时反馈
- 模式切换后立即刷新表格
- 状态栏显示当前模式和匹配数
- 视觉标记低于阈值的匹配
