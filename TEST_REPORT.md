## 已知问题

### Windows 控制台编码
- **问题**: Windows 控制台使用 GBK 编码，无法显示某些 Unicode 字符
- **影响**: 控制台输出可能显示乱码
- **解决方案**: 所有日志使用 UTF-8 写入文件（logs/app.log）
- **状态**: ✅ 已解决，不影响功能
